# Security Policy

## Supported Versions

Only the latest version of `lineclientbot` is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >=0.1.1 | :white_check_mark: |
| < 0.1.1 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public issue. Instead, report it privately to the maintainer:

- Email: [chayangkoon94@gmail.com]

We aim to acknowledge receipt of your report within 48 hours, and to provide a detailed resolution plan within 7 days.
